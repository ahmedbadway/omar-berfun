# Hero background video

Place the hero background video here as `hero-bg.mp4`.

It is loaded by `src/sections/HeroSection.jsx` via
`import.meta.env.BASE_URL + 'videos/hero-bg.mp4'` and plays fullscreen,
autoplay / muted / loop / playsInline, with a dark overlay on top.

Recommended: 1080p (or 720p) H.264 MP4, short seamless loop, optimized
file size (a few MB) for fast loading on GitHub Pages.
